## DESCOBERTA-CHAVE DO ESTUDO (muda o modelo mental das otimizacoes antigas)
As imagens do jogo NAO usavam o wrapper 16-bit na textura residente! O gl2
premultiplica na GPU: upload TEMPORARIO (esse sim passava no tex16/downscale)
-> passe FTL num FBO -> textura final nasce de glCopyTexImage2D FULL-SIZE
RGBA8888 (fora do wrapper). Ou seja: cada imagem morava na RAM a 32bpp em
tamanho cheio. O ganho de fps do tex16 vinha so do upload/FTL menores.
Cadeia da trava: residentes 8888 full + rajadas de predict + render-textures
de dissolve + texto por linha + ciclos Render<->GLTexture esperando GC
(thresholds altos seguram lixo qdo se clica rapido, sem frame idle)
-> 852MB -> swap na SD -> thrash -> congela ate o sshd.

## O QUE FOI IMPLEMENTADO (local, compilado, validado no host)
1. **Caminho premultiplied direto** (im.py, fim do arquivo,
   _ss_install_premul_path): monkeypatch em Texture.load (classe Python do
   gl2texture) -> load_gltexture_premultiplied() = UM upload interceptavel.
   Texto (ja premultiplied) e video (video.py marca ss_orig_path) ficam como
   eram. Aviso pro C via ctypes ss_tex_hint(1)/(0) (exportado, -rdynamic).
2. **ETC1 dupla-camada em runtime** (src/etc1.c novo + src/imports.c):
   - encoder ETC1 clean-room (~250 linhas), validado com decoder independente
     + PSNR no host (solid 38.7dB, gradiente 40dB, mascara-alpha 37dB);
   - no upload com hint: premultiply em C + downscale (TEXDS_MAX) + RGB vira
     ETC1 4bpp (glCompressedTexImage2D 0x8D64); se tem alpha, canal A vira
     textura GEMEA ETC1 cinza (g_tex_twin). 32bpp -> 8bpp c/ alpha, 4bpp opaco;
   - shader remendado no wrap_glShaderSource: texture2D(texN, -> ss_texN( +
     preludio com u_ss_dualN/u_ss_atexN (INVISIVEL pro Ren'Py, que so parseia
     o proprio fonte). Validado com glslangValidator (texture/blur/dissolve);
   - predraw (wrap_glDrawArrays/Elements): rastreia glUniform1i/ActiveTexture/
     BindTexture -> sabe o que esta em tex0..2, seta u_ss_dualN e liga a gemea
     na unidade ALTA (MAX_UNITS-1-s = 7/6/5);
   - TexSubImage em textura ETC1 = ignorado (contador g_etc1_sub_skips);
     glDeleteTextures/glCopyTexImage2D limpam gemea/registro.
3. **GC periodico** (config.py, fim): gc.collect() completo a cada
   SUMMERTIME_GC_EVERY interacoes (default 8; 0 desliga) via
   start_interact_callbacks — solta os ciclos Render/GLTexture que seguram
   textura morta quando se clica rapido.
4. **Receita publica** atualizada: facilitando_o_trabalho/receitas/
   08-texturas-etc1-etc2.md (secoes 6-8: teoria dupla camada, encoder runtime,
   caso Ren'Py/premultiply).
